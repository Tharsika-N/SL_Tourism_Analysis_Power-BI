## Overview
This project analyzes monthly tourist arrivals to Sri Lanka from 2019 to 2024 using Power BI.

## Data Source
Sri Lanka Tourism Development Authority (SLTDA)

## Key Steps
- Combined multiple Excel files using Power Query (Folder method)
- Cleaned and standardized inconsistent formats
- Created Fact and Dimension tables
- Built relationships for time-based analysis
- Created KPIs using DAX

## KPIs Created
- Total Tourist Arrivals
- Average Monthly Arrivals
- Maximum Arrivals
- Year-over-Year (YoY) Growth %
  
## Tools Used
- Power BI
- Power Query
- DAX
- Excel

## Dashboard Features
- Interactive visuals and charts for trend analysis
- Filters and slicers for time periods and categories
- KPI cards highlighting key performance metrics
